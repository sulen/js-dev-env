# JavaScript development environment
### Ready to go enviroment that handles transpiling, bundling, testing and npm scripts for automation.

## Tools used
- Package Manager: npm
- Automation: npm scripts
- Transpiling: Babel
- Bundling: Webpack
- Linting: ESLint
- Testing: Jest
- Mocking: json-server, JSON Schema Faker
